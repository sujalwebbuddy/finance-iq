<h2>Overview</h2>
<p>FinanceIQ is a comprehensive, production-ready financial management application that helps users track expenses, manage budgets, and process receipts with AI. Built with React 19, Node.js, and modern web technologies, this application provides a seamless experience for personal and business financial management. The platform features a modern, responsive design and includes everything needed to run a financial management SaaS business, from user authentication to Stripe payment processing.</p>

<h2>Key Features</h2>
<h3>AI-Powered Receipt Processing</h3>
<p>Automatically extract transaction details from receipts using Google Gemini AI. Users can upload receipt images and the AI intelligently identifies merchant names, amounts, dates, and transaction categories. The application processes receipts in seconds and eliminates manual data entry.</p>
<h3>Comprehensive Transaction Management</h3>
<p>Track all income and expenses with detailed categorization and tagging. The application supports multiple transaction types:</p>
<ul>
<li>Income and expense tracking with custom categories</li>
<li>Multi-currency support for international transactions</li>
<li>Transaction search and filtering capabilities</li>
<li>Receipt attachment and digital document management</li>
<li>Real-time balance calculations and financial summaries</li>
</ul>
<h3>Smart Budget Planning</h3>
<p>Create and manage budgets with intelligent spending limits and alerts. Features include:</p>
<ul>
<li>Category-based budget allocation</li>
<li>Spending limit alerts and notifications</li>
<li>Budget vs. actual spending comparisons</li>
<li>Budget performance analytics and insights</li>
<li>Flexible budget periods (weekly, monthly, yearly)</li>
</ul>
<h3>Recurring Transactions</h3>
<p>Automate recurring income and expenses with flexible scheduling options. The system automatically generates transactions based on predefined rules and supports various frequencies:</p>
<ul>
<li>Daily, weekly, monthly, and yearly recurring transactions</li>
<li>Custom intervals and end dates</li>
<li>Automatic transaction generation and notifications</li>
<li>Recurring transaction analytics and forecasting</li>
</ul>
<h3>User Management</h3>
<p>Complete user management system with the following features:</p>
<ul>
<li>User authentication with JWT and Google OAuth</li>
<li>User dashboard for managing financial data</li>
<li>Transaction and budget history tracking</li>
<li>Multi-currency support and preferences</li>
<li>User profile and settings management</li>
<li>Secure session management and password reset</li>
</ul>
<h3>Subscription System</h3>
<p>Flexible subscription-based pricing system with four tiers:</p>
<ul>
<li>Free Plan - 50 transactions, 3 budgets, 2 recurring transactions</li>
<li>Basic Plan - $9.99/month - 500 transactions, 10 budgets, 10 receipts, AI receipt processing</li>
<li>Pro Plan - $19.99/month - Unlimited transactions, advanced analytics, API access, priority support</li>
<li>Enterprise Plan - $49.99/month - All Pro features plus SSO, white-label options</li>
</ul>
<p>The free plan allows users to test core features. Paid plans unlock advanced AI capabilities and higher usage limits.</p>
<h3>Payment Processing</h3>
<p>Integrated Stripe payment system for secure subscription management. The application handles checkout sessions, webhook processing, automatic subscription activation, and plan upgrades/downgrades with proration.</p>
<h3>Financial Analytics Dashboard</h3>
<p>Users can analyze their financial data through comprehensive dashboards:</p>
<ul>
<li>Interactive charts showing income vs. expenses over time</li>
<li>Budget performance tracking with visual indicators</li>
<li>Spending category breakdowns and trends</li>
<li>Financial goal progress monitoring</li>
<li>Export capabilities for tax preparation and reporting</li>
</ul>
<h3>Usage Tracking & Limits</h3>
<p>Built-in usage tracking and limits based on subscription plans to ensure fair resource allocation. The system tracks transactions, receipts, budgets, and API requests with real-time monitoring and alerts.</p>
<h2>Technical Specifications</h2>
<h3>Technology Stack</h3>
<ul>
<li>Frontend: React 19 with Vite build system</li>
<li>Backend: Node.js with Express.js framework</li>
<li>Language: JavaScript (ES6+)</li>
<li>UI Library: Tailwind CSS with custom components</li>
<li>Database: MongoDB with Mongoose ODM</li>
<li>Authentication: JWT with Passport.js (Google OAuth)</li>
<li>Payments: Stripe with webhook integration</li>
<li>AI Integration: Google Gemini API for receipt processing</li>
<li>File Upload: Multer with validation</li>
<li>Charts: Chart.js with React Chart.js 2</li>
<li>Notifications: React Toastify</li>
<li>HTTP Client: Axios</li>
</ul>
<h3>System Requirements</h3>
<ul>
<li>Node.js 18 or higher</li>
<li>npm or yarn package manager</li>
<li>MongoDB database (MongoDB Atlas recommended)</li>
<li>API keys for required services</li>
</ul>

Check MongoDB Atlas pricing for details.</p>
<h4>Stripe</h4>
<p>Stripe processes subscription payments. Stripe charges a transaction fee of 2.9% plus $0.30 per successful subscription payment. There are no monthly fees for standard accounts. Visit Stripe pricing page for complete fee structure.</p>
