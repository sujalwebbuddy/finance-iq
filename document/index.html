Welcome to FinanceIQ

Version 1.0
Updated November 2025
Welcome to FinanceIQ v1.0! This comprehensive documentation will guide you through setting up and deploying your personal finance management application. Track income, expenses, budgets, receipts, and visualize your financial data with powerful analytics.

Overview

FinanceIQ is a comprehensive personal finance management application that helps you track income, expenses, budgets, and receipts. With AI-powered receipt scanning, recurring transaction management, and detailed analytics, FinanceIQ makes financial management simple and intuitive.

Transaction Management

Track income and expenses with custom categories. Add, edit, and delete transactions with ease. View your financial summary at a glance.

AI Receipt Scanning

Upload receipts and automatically extract expense details using Google Gemini OCR. Create transactions instantly from receipt images.

Budget Management

Create and manage budgets for different categories. Track spending against budgets and get alerts when limits are exceeded.

Recurring Transactions

Set up recurring income and expenses with daily, weekly, monthly, or annual frequencies. Never miss a payment or income again.

Analytics & Charts

Visualize your financial data with interactive charts. View category breakdowns, income vs expense trends, and spending patterns.

Multi-Currency Support

Support for multiple currencies including USD, EUR, GBP, JPY, CAD, AUD, and INR. Switch currencies seamlessly.

System Architecture

Tech Stack:
  • Frontend: React 19 with Vite
  • Styling: Tailwind CSS
  • Charts: Chart.js with react-chartjs-2
  • Backend: Node.js with Express
  • Database: MongoDB with Mongoose (MongoDB Atlas recommended)
  • Authentication: JWT with Google OAuth (Passport.js)
  • Payments: Stripe (Subscription-based system)
  • AI Integration: Google Gemini API for OCR receipt scanning
  • File Upload: Multer for receipt image uploads
  • Deployment: Frontend on Netlify, Backend on Render

Key Features

Core Functionality

  • Transaction Management: Add, edit, and delete income and expense transactions with custom categories
  • Receipt Scanning: Upload receipt images and automatically extract transaction details using Google Gemini OCR
  • Budget Tracking: Create budgets for categories and track spending against limits
  • Recurring Transactions: Set up recurring income and expenses with flexible frequency options
  • Financial Analytics: View income vs expense charts, category breakdowns, and spending trends
  • Multi-Currency Support: Support for USD, EUR, GBP, JPY, CAD, AUD, and INR

Advanced Features

  • Subscription Plans: Free, Basic, Pro, and Enterprise tiers with different feature access
  • Usage Limits: Plan-based limits on transactions, receipts, budgets, and recurring transactions
  • Dashboard Overview: Comprehensive financial summary with charts and recent transactions
  • Category Management: Create and manage custom transaction categories
  • User Settings: Manage profile, currency preferences, and account deletion
  • Mobile Responsive: Fully responsive design for all devices

Prerequisites

Before you begin, ensure you have:
  • Node.js 18+ installed
  • MongoDB database - MongoDB Atlas recommended (free tier available)
  • Google Gemini API key (for receipt OCR scanning)
  • Stripe account for subscription payments
  • Google OAuth credentials (optional, for Google login)

Quick Start

  1. Clone: Get the repository from GitHub
  2. Backend Setup: Navigate to backend/ folder, run npm install, and configure .env file
  3. Frontend Setup: Navigate to frontend/ folder, run npm install, and configure .env file
  4. Database: Set up MongoDB Atlas connection string
  5. Start Backend: Run npm run dev in backend folder (runs on port 5000)
  6. Start Frontend: Run npm run dev in frontend folder (runs on port 5173)
  7. Deploy: Follow deployment guides for Netlify (frontend) and Render (backend)
Pro Tip: For production use, Netlify is recommended for frontend deployment with automatic builds. Render is recommended for backend deployment with free tier available.
