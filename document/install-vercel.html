Install on Vercel

⚠️ Deprecated: This page is for the old application. Paisable uses separate frontend and backend deployments. Please see How to install on Netlify for frontend deployment.
1. Ensure your project is pushed to GitHub (see How to push to GitHub guide).
2. In Vercel, click New Project → Import your repository.
3. Framework preset: Next.js. Keep default build command next build and output.
4. Add all environment variables from the Environment Variables page.
5. Deploy. After deploy, update NEXT_PUBLIC_APP_URL with your production domain (e.g., https://your-domain.vercel.app).

Notes

  • Next.js App Router is already configured. No further runtime settings are needed.
  • Stripe webhooks must be configured post-deploy. See Stripe keys.
  • MongoDB connection must be configured in environment variables.
  • All API routes are served from the same Next.js application under /api.
This is a monolithic Next.js 14 application. All frontend and backend code runs in a single Next.js server. No separate backend deployment is needed.
