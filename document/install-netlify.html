Install Frontend on Netlify

Prerequisite: Make sure you've completed the How to push to GitHub and deployed your backend to Render (see How to install on Render) before deploying the frontend.

Deploying Frontend to Netlify

1. Ensure your project is pushed to GitHub (see How to push to GitHub guide).
2. Sign up or log in to Netlify.
3. Click Add new site ‚Üí Import an existing project.
4. Connect to GitHub and select your repository.
5. Configure build settings:
  • Base directory: frontend
  • Build command: npm run build
  • Publish directory: frontend/dist
6. Add environment variables:
  • VITE_API_URL - Your backend API URL (e.g., https://your-backend.onrender.com/api)
  • VITE_APP_NAME - App name (optional)
  • VITE_APP_DESCRIPTION - App description (optional)
7. Click Deploy site.
8. After deployment, update your backend FRONTEND_URL environment variable with your Netlify domain.

Configuration Details

  • Framework: Vite (detected automatically by Netlify)
  • Build Command: npm run build (runs from frontend directory)
  • Output Directory: dist (Vite's default build output)
  • Node Version: Netlify automatically detects from package.json

Environment Variables

Add these in Netlify Dashboard ‚Üí Site settings ‚Üí Environment variables:

VITE_API_URL=https://your-backend.onrender.com/api
VITE_APP_NAME=FinanceIQ
VITE_APP_DESCRIPTION=Personal finance management

Custom Domain (Optional)

  1. Go to Site settings ‚Üí Domain management
  2. Click "Add custom domain"
  3. Enter your domain name
  4. Follow DNS configuration instructions

Notes

  • Netlify automatically builds and deploys on every push to your main branch
  • Preview deployments are created for pull requests
  • Make sure your backend is deployed and accessible before deploying frontend
  • Update FRONTEND_URL in backend environment variables after frontend deployment
