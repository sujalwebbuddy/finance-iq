User Guide

Complete User Guide: Learn how to use FinanceIQ v1.0, from setting up your account to managing transactions, budgets, receipts, and analyzing your financial data.

Getting Started

1. Account Creation

  1. Sign Up: Click "Get Started" or "Sign Up" on the homepage
  2. Choose Method: Sign up with email/password or Google OAuth
  3. Initial Setup: Complete the setup wizard to choose your default currency
  4. Dashboard Access: Automatically redirected to your dashboard after setup

2. Understanding Subscription Plans

Plan System:
  • Free Plan: 50 transactions, 3 budgets, 2 recurring transactions (no receipt OCR), exports: none
  • Basic Plan: 500 transactions, 10 budgets, 10 recurring transactions, 10 receipt scans, exports: 5 per month
  • Pro Plan: Unlimited transactions, budgets, recurring transactions, and receipt scans, exports: unlimited
  • Enterprise Plan: All Pro features plus SSO and white-label options, exports: unlimited

üí∞ Managing Transactions

Adding a Transaction

  1. Navigate: Go to the Transactions page from the sidebar
  2. Click Add: Click the "Add Transaction" button
  3. Select Type: Choose Income or Expense
  4. Enter Details:
    • Amount: Enter the transaction amount
    • Category: Select or create a custom category
    • Description: Add a description (optional)
    • Date: Select the transaction date
  5. Save: Click "Save" to add the transaction
Pro Tip: Use consistent category names to better track your spending patterns. You can create custom categories for specific needs.

Editing and Deleting Transactions

  • Edit: Click on any transaction to open the edit modal
  • Update: Modify the transaction details and click "Save"
  • Delete: Click the delete button to remove a transaction permanently

Managing Categories

  1. Access Categories: Click "Manage Categories" from the transactions page
  2. Add Category: Enter a new category name and click "Add"
  3. Delete Category: Click the delete icon next to unused categories

üìä Budget Management

Creating a Budget

  1. Navigate: Go to the Budgets page from the sidebar
  2. Click Add: Click "Create Budget" or the "+" button
  3. Set Budget Details:
    • Category: Select the expense category
    • Amount: Enter your budget limit
    • Period: Choose monthly, weekly, or custom period
  4. Save: Click "Save" to create the budget
Budget Limits:
  • Free plan allows up to 3 budgets
  • Basic plan allows up to 10 budgets
  • Pro and Enterprise plans have unlimited budgets

Tracking Budget Progress

  • Visual Progress: View progress bars showing spending vs. budget
  • Alerts: Get notified when approaching or exceeding budget limits
  • History: Review budget performance over time

üßæ Receipt Scanning

Uploading and Scanning Receipts

  1. Navigate: Go to the Receipts page from the sidebar
  2. Upload Receipt: Click "Upload Receipt" and select an image file
  3. AI Processing: Google Gemini OCR automatically extracts transaction details
  4. Review: Review the extracted information (amount, date, merchant)
  5. Confirm: Confirm or edit the details and save as a transaction
Receipt OCR: The system uses Google Gemini AI to extract transaction details from receipt images. Supported formats include JPG, PNG, and PDF. Ensure receipts are clear and well-lit for best results.

üîÑ Recurring Transactions

Setting Up Recurring Transactions

  1. Navigate: Go to the Recurring Transactions page
  2. Click Add: Click "Add Recurring Transaction"
  3. Configure:
    • Name: Enter a descriptive name (e.g., "Netflix Subscription")
    • Type: Select Income or Expense
    • Category: Choose the appropriate category
    • Amount: Enter the transaction amount
    • Frequency: Select daily, weekly, monthly, or annually
    • Start Date: Choose when the recurring transaction should begin
  4. Save: Click "Save" to create the recurring transaction

Managing Recurring Transactions

  • View All: See all your recurring transactions in one place
  • Edit: Modify frequency, amount, or other details
  • Delete: Remove recurring transactions you no longer need
  • Next Due: View when each recurring transaction is next due

Troubleshooting

Common Issues

Frequent Problems:
  • Transaction Limit Reached: Upgrade your plan to add more transactions
  • Receipt Upload Failed: Check file size (max 10MB) and format (JPG, PNG)
  • OCR Extraction Failed: Ensure receipt image is clear and well-lit
  • Budget Limit Reached: Upgrade plan or delete unused budgets
  • Login Issues: Clear browser cache and cookies, or try Google OAuth

Error Messages

Error Cause Solution
Transaction Limit Exceeded Reached maximum transactions for your plan Upgrade to a higher plan or wait for monthly reset
Receipt Upload Failed File too large or invalid format Use JPG or PNG format, max 10MB per file
OCR Processing Failed Receipt image unclear or unsupported format Upload a clear, well-lit receipt image
Budget Limit Reached Maximum number of budgets for your plan Upgrade plan or delete unused budgets
Payment Failed Stripe payment processing error Check payment method and try again
