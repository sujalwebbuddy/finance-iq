  // ... more configuration

2. Logo and Images

  • Images: Add custom images to public/ directory

Build and Deployment

Development

Run the development server:

cd frontend
npm run dev

# The application runs on http://localhost:5173
# API calls are made to backend at http://localhost:5000/api

Production Build

cd frontend

# Build for production
npm run build

# Preview production build
npm run preview

# The build output is in the dist/ directory
Note: This is a React + Vite application. The frontend is separate from the backend and communicates via REST API. Build output is static files that can be deployed to Netlify, Vercel, or any static hosting service.

Environment Variables

Create .env in the frontend directory:

# Frontend Environment Variables (in frontend/.env)
VITE_API_URL=http://localhost:5000/api

# App Configuration (Optional)
VITE_APP_NAME=Paisable
VITE_APP_DESCRIPTION=Personal finance management

Troubleshooting

Common Issues:
  • API Connection Failed: Check VITE_API_URL is set correctly in frontend/.env
  • Build Errors: Check for JavaScript/JSX errors and missing dependencies
  • Styling Issues: Ensure Tailwind classes are included in content paths in tailwind.config.js
  • Route Not Working: Verify React Router routes are configured correctly
  • Authentication Issues: Check AuthContext and token storage
  • CORS Errors: Ensure backend CORS is configured to allow frontend origin
